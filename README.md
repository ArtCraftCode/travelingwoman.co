# travelingwoman.co

Website for customizing Traveling Woman, now with Ravelry authentication.

## Development

**Assets**

- node v5.1.0 (`nvm use 5`)
- run `npm start` to watch and compile

**Server**

- Ruby 2.3.1
- `bundle exec app.rb`
- requires `ENV['RAV_ACCESS']`, `ENV['RAV_SECRET']`
